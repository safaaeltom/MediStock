# MediStock 💊

A simple pharmacy inventory management application built with **HTML, CSS, and JavaScript**.

---

## 📌 About the Project

MediStock is a learning project focused on building a practical pharmacy inventory system while improving my JavaScript skills.

The application allows users to manage a list of medicines, track quantities, and update inventory dynamically in the browser.

This project is still **in active development** and will continue to evolve as I learn more JavaScript and web development concepts.

---

## 🚧 Current Features

* Add new medicines to the inventory
* Display list of available medicines
* Track medicine quantities
* Delete medicines from the list
* Basic UI styling for inventory layout
* Dynamic DOM updates using JavaScript

---

## 🧠 Technologies Used

* HTML
* CSS
* JavaScript (Vanilla)
